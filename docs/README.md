# Ordmorf — Kunskapsbakgrund

Denna mapp innehåller pedagogisk och vetenskaplig bakgrund som ligger till grund för Ordmorf-spelets design. Materialet används av Claude som kontext vid vidareutveckling.

## Struktur

```
docs/
├── README.md              ← denna fil
├── research/              ← forskningsrapporter och analyser
│   ├── 01-morfologi-klassrummet.md    ← huvudrapport
│   └── 02-morfologi-f6-research.md    ← systematisk research F-6
└── design/                ← designbeslut och pedagogiska principer
    └── exercise-principles.md         ← regler för övningskonstruktion
```

## Så här lägger du till nytt material

1. Dela texten med Claude i en konversation
2. Be Claude lägga den i rätt underkatalog med nästa löpnummer
3. Pusha till GitHub: `git add docs/ && git commit -m "Ny källa: [titel]" && git push`
